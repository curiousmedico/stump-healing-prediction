**Machine Learning Model Development and Interpretability**

A gradient boosting classifier (XGBoost) was developed to predict 30-day stump 
healing outcomes using clinical and biochemical parameters as features. The model 
was trained on [N=xxx] patients with [xxx] clinical variables extracted from the 
institutional records. Hyperparameter optimization was performed using [specify: 
grid search/Bayesian optimization] with 5-fold cross-validation. Model performance 
was evaluated on a held-out test set (20% of data) using standard metrics including 
accuracy, sensitivity, specificity, and receiver operating characteristic (ROC) 
area under the curve (AUC).

To ensure model interpretability and validate clinical relevance, SHAP 
(SHapley Additive exPlanations) analysis was applied to quantify the contribution 
of each clinical and biochemical variable to stump healing predictions. SHAP values 
were computed using TreeExplainer, optimized for tree-based models, on the complete 
test cohort. Global feature importance was assessed via mean absolute SHAP values, 
identifying which variables had the greatest influence on healing outcomes across 
all patients. Individual patient-level explanations were generated to provide 
personalized interpretability, enabling clinicians to understand the specific factors 
driving each prediction. This analysis validated that the model prioritizes the 
biochemical parameters of the Amit Jain Stump Healing Model (serum sodium, serum 
creatinine, and HDL cholesterol), as well as referral delay, confirming alignment 
between the AI model and established clinical knowledge. The combination of predictive 
performance and interpretability ensures "white box" transparency suitable for 
clinical adoption in the surgical setting.
